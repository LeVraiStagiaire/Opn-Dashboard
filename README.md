
# Opn Dashboard

**Opn Dashboard** est une **application web légère en HTML et JavaScript pur** qui permet de créer un **tableau de bord personnalisé** avec des widgets dynamiques : horloge, météo, calendrier, flux d’informations et plus encore.

Idéal pour un écran d’accueil, un tableau de suivi ou simplement un projet communautaire modulaire.

---

## 🚀 Fonctionnalités principales

* Gestion de widgets modulaires : ajouter, supprimer, déplacer
* Widgets intégrés :
  * Horloge digitale/analogique
  * Météo locale
  * Calendrier et événements
* Interface responsive et personnalisable
* Compatible avec Chrome, Firefox et navigateurs modernes
* Léger et sans dépendances externes
* Support communautaire pour les nouveaux widgets

## 📦 Installation rapide

### Cloner le dépôt

```bash
git clone https://github.com/LeVraiStagiaire/Opn-Dashboard.git
cd Opn-Dashboard
```

### Lancer localement

* Ouvrir `index.html` dans votre navigateur
* Ou utiliser un serveur local recommandé (ex. **Live Server** sur VSCode)

## 🛠 Structure du projet

- `/widgets` → widgets individuels
- `/index.html` → page principale
- `/script.js` → script principal
- `/style.css` → styles principaux
